Para actualizar un paciente :
- Primero se realiza la busqueda segun el criterio que se elija dni, apellido, mail
- Luego el formulario se llena con los valores de ese paciente
- Cambio el valor que se necesite y se da click al boton editar

OBSERVACIONES:
- Una vez enviado el PUT nos va a devolver un mensaje cannot Get /actualizar
- El paciente se va a modificar igualemente en la base de datos
- Solo se puede buscar por dni y por mail

Para cargar un paciente:

- Momentaneamente para que funcione es necesario "comentar desde la linea 5 a 8
- Se llena el formulario y se da a cargar
